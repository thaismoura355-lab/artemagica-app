# Artemagica App

## Project Description

Artemagica is a React Native drawing and coloring app designed specifically for Android devices. It provides users with a limitless canvas and a variety of tools to unleash their creativity. With basic features such as brush settings, color palettes, and the ability to save and share artwork, users of all ages can enjoy the art-making process.

### Features:
- **Intuitive Drawing Tools:** Offers diverse brushes and adjustable sizes for personalized design.
- **Vibrant Color Palettes:** Users can choose from a wide array of colors or create their own custom palettes.
- **Layer Management:** Supports multi-layer editing to enhance complexity in artwork.
- **Save & Share Artwork:** Users can save their creations locally or share them on social media platforms.
- **Offline Capabilities:** No internet connection is required to create and edit artwork.

### Technologies Used:
- **React Native:** The core framework for building the mobile app, ensuring a smooth experience on Android devices.
- **Redux:** For state management, allowing a seamless flow of data across the app.
- **Expo:** To expedite development and simplify the testing process.

### Installation:
To get started with Artemagica:
1. Clone the repository: `git clone <repository-url>`
2. Navigate to the project directory: `cd artemagica-app`
3. Install the dependencies: `npm install`
4. Run the app: `npm start`
